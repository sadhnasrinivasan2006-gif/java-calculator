# Java Swing Calculator

A simple calculator application built with Java Swing, supporting basic arithmetic operations (addition, subtraction, multiplication, division) to help users with quick computations via a graphical interface[1][2].

## Features

- User-friendly interface for easy calculations[2].
- Supports addition, subtraction, multiplication, and division[1][2].
- Responsive design: Adjusts to various screen sizes[2].
- Clear button to reset all input[1][2].
- Real-time arithmetic calculations[2].

## Getting Started

### Prerequisites

- Java Development Kit (JDK) 8 or above installed on your system[2].
- A Java IDE such as Eclipse or IntelliJ IDEA (optional but recommended)[1].

### Installation

1. Clone this repository:
   ```
   git clone https://github.com/sadhnasrinivasan2006-gif/calculator.git
   ```
2. Open the project in your Java IDE[1].
3. Compile and run `Calculator.java`:
   ```
   javac Calculator.java
   java Calculator
   ```

## Usage

- Use number buttons (0-9) to enter values.
- Use operator buttons (+, -, *, /) for the desired calculation.
- Press `=` to view the result.
- Use the `CLEAR` button to reset input and start a new calculation.

## Project Structure

- `Calculator.java`: Main implementation using JFrame, ActionListener, and Swing components.
